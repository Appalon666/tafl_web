# Tafl (Hnefatafl) — Web

Цифровая адаптация древней скандинавской стратегии **Тафл / Hnefatafl**
(«шахматы викингов») для веба.

## Об игре

Асимметричная игра для двух сторон:

- **Атакующие** — многочисленная армия, цель — окружить короля.
- **Защитники** — король и немногочисленная стража в центре; цель —
  провести короля в один из четырёх угловых полей.

Захват фигуры — зажатием с двух сторон. Правила простые, стратегия глубокая.
Главная фишка — асимметрия сторон.

## Статус

Старт проекта. Движок: Godot (как в [FaG](https://github.com/Appalon666/FaG)).

## Структура (планируемая)

```
scenes/      — сцены Godot
scripts/     — игровая логика, ИИ, UI
assets/      — спрайты, звук, шрифты
localization/— переводы (ru/en)
```
