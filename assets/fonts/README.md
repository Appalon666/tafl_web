# UI-шрифт

Сюда кладётся шрифт всего интерфейса. Подключение уже прописано в коде
(`scripts/systems/Settings.gd` → `ThemeDB.fallback_font`): если файл есть — он
становится глобальным шрифтом UI; если файла нет — игра молча использует дефолтный
шрифт Godot (без ошибок).

## Как подключить (3 шага)

1. Скачай один из шрифтов ниже (формат `.ttf` или `.otf`).
2. Переименуй файл в **`UIFont.ttf`** и положи его в эту папку (`assets/fonts/`).
3. Открой проект в Godot 4.7 — он импортирует шрифт автоматически. Готово, весь
   текст (включая кириллицу) пойдёт этим шрифтом и в браузерной сборке.

> Путь и имя должны быть точно `res://assets/fonts/UIFont.ttf`. Хочешь другое имя —
> поменяй `UI_FONT_PATH` в `Settings.gd`.

## Рекомендованные бесплатные шрифты (полная кириллица, лицензия OFL)

Для основного текста UI нужна **читаемость + кириллица**, поэтому берём чистый
гротеск, а не рунический декоративный (у рунических обычно нет кириллицы):

- **Inter** — современный UI-шрифт, отличная кириллица.
  https://fonts.google.com/specimen/Inter
- **Noto Sans** — универсальный, полное покрытие кириллицы.
  https://fonts.google.com/noto/specimen/Noto+Sans
- **PT Sans** — изначально спроектирован под кириллицу (ParaType).
  https://fonts.google.com/specimen/PT+Sans

## Заголовок «TAFL» (опционально, скандинавский акцент) — уже подключён

Подключение для логотипа главного меню **уже прописано** (`MainMenu.gd` берёт
`Settings.title_font`). Логотип — латиница «TAFL», поэтому кириллица тут не нужна,
можно взять декоративный «нордический» шрифт:

1. Скачай декоративный шрифт, например **Norse** (Joël Carrouché, бесплатный):
   https://www.dafont.com/norse.font
2. Переименуй в **`TitleFont.ttf`** и положи сюда (`assets/fonts/`).
3. Открой проект в Godot — заголовок «TAFL» пойдёт этим шрифтом, а весь остальной
   текст останется читаемым `UIFont.ttf`.

Файла нет → заголовок просто рисуется обычным шрифтом, без ошибок.

## Важно для веб-экспорта

Файл шрифта попадает в сборку как ресурс. Убедись, что в пресете экспорта он не
исключён фильтром (по умолчанию включается). Рендер — GL Compatibility, шрифт
растеризуется на стороне Godot, так что в браузере отобразится идентично.
