## ⚡ TL;DR — что проверять ПЕРЕД каждой отправкой

- [ ] **Реклама ставит игру на паузу** (геймплей + звук) на `onOpen`, снимает на `onClose`. (п. 4.7) ← код
- [ ] Реклама показывается только в **логических паузах**, не мешает игре. (п. 4.4)
- [ ] **Скриншоты:** реальный геймплей ≥ 70% площади. (п. 5.1.1.2)
- [ ] **Видео:** реальный геймплей ≥ 70% длительности. (п. 5.1.1.3)
- [ ] **Промо/медиа** — только элементы из самой игры, без внешних образов. (п. 5.1.1)
- [ ] Скриншот игры **нельзя** использовать как иконку/обложку. (п. 5.6)
- [ ] **Иконка/обложка:** без скруглённых углов и рамок. (п. 8.3.3)
- [ ] Медиа без артефактов сжатия/теней/обрезанного текста. (п. 8.3.1)
- [ ] Медиа без системного UI (статус-бар, батарея) и без UI Яндекс.Игр. (п. 8.3.4)
- [ ] **Все языковые поля** реально переведены (поле `[ru]` — по-русски, `[en]` — по-английски). (п. 8.2.3)
- [ ] **Название игры** одинаково в игре и во всех материалах драфта, для каждого языка. (п. 5.1.3)
- [ ] **Ключевые слова** соответствуют игре (не лепить нерелевантные жанры). (п. 5.4)

## 🧩 Повторяющиеся код-грабли (Godot + Yandex SDK)

### Пауза при рекламе (п. 4.7) — обязательный паттерн

В JS-обёртке SDK (вызов из Godot через JavaScriptBridge):

```javascript
ysdk.adv.showFullscreenAdv({
    onOpen:  () => { /* пауза игры + mute */ },
    onClose: (wasShown) => { /* resume + unmute */ },
    onError: (e) => { /* тоже снять паузу, чтобы не залипнуть */ }
});

ysdk.adv.showRewardedVideo({
    onOpen:     () => { /* пауза + mute */ },
    onRewarded: () => { /* выдать награду */ },
    onClose:    (wasShown) => { /* resume + unmute */ },
    onError:    (e) => { /* снять паузу */ }
});
```

В Godot-стороне (`YandexSDK.gd`):
- `onOpen`  → `get_tree().paused = true`, приглушить `AudioManager`.
- `onClose` / `onError` → `get_tree().paused = false`, вернуть звук.
- ⚠️ Узлы, которые должны продолжать жить на паузе (сам SDK-мост), ставить
  `process_mode = PROCESS_MODE_ALWAYS`.
- ⚠️ Обязательно обрабатывать `onError`/таймаут — иначе при сбое рекламы игра
  останется на паузе навсегда.

## 📚 Полный свод требований (выжимка из официальной документации)

### Реклама
- **4.4** — реклама не мешает игре, показ только в логических паузах.
- **4.7** — при полноэкранной рекламе (интерстишел/rewarded) **звук и геймплей на паузе**.

### Промо и медиа
- **5.1.1.2** — скриншоты: реальный геймплей ≥ 70% кадра.
- **5.1.1.3** — видео: реальный геймплей ≥ 70% длительности.
- **5.1.3** — название игры идентично в игре и всех материалах драфта (для каждого языка).
- **5.4** — теги и ключевые слова соответствуют теме и содержанию.
- **5.6** — скриншоты игры нельзя использовать как иконки/обложки.

### Формат медиа
- **8.3.1** — хорошее качество, без артефактов сжатия, лишних теней, обрезанного текста.
- **8.3.3** — без рамок и скруглённых углов.
- **8.3.4** — без системного UI (статус-бар, индикатор батареи) и без UI Яндекс.Игр.

### Тексты и перевод
- **8.2.3** — все языкозависимые тексты переведены на соответствующий язык.
